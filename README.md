# Luxury Portfolio Website

A sophisticated 3-page portfolio website featuring elegant design, smooth animations, and premium aesthetics.

## Features

### 🎨 **Luxury Design**
- Premium color scheme with gold accents
- High-quality Unsplash background images
- Elegant typography with Playfair Display and Montserrat fonts
- Glassmorphism effects with backdrop filters

### 🎭 **Animations & Effects**
- Smooth page transitions
- Fade-in and slide-in animations
- Typewriter effect for subtitles
- Hover effects on cards and buttons
- Skill bar animations
- Parallax scrolling effects
- Testimonial slider with auto-rotation

### 📄 **Pages**

#### 1. **Home Page** (`index.html`)
- Hero section with gradient overlay
- About me section with professional portrait
- Services showcase with hover effects
- Responsive design

#### 2. **Qualifications Page** (`qualifications.html`)
- Education timeline with alternating layout
- Interactive skill bars with shimmer effects
- Certifications showcase
- Professional qualifications display

#### 3. **Projects Page** (`projects.html`)
- Project gallery with alternating layouts
- Image zoom effects on hover
- Technology stack display
- Client testimonials slider

### 🛠 **Technologies Used**

- **HTML5** - Semantic structure
- **CSS3** - Advanced styling and animations
- **JavaScript** - Interactivity and dynamic effects
- **Google Fonts** - Playfair Display & Montserrat
- **Unsplash API** - High-quality background images

## 🚀 **How to Use**

1. **Clone the repository** or download the files
2. **Open `index.html`** in your browser to start
3. **Navigate** between pages using the navigation menu
4. **Customize** content by editing the HTML files
5. **Replace** placeholder images with your own

## 🎯 **Customization**

### Personal Information
- Edit `index.html` to update your name, bio, and services
- Update `qualifications.html` with your education and skills
- Add your projects to `projects.html`

### Styling
- Modify colors in `css/style.css` (CSS variables at the top)
- Adjust animations in `css/animations.css`
- Change fonts in the HTML head section

### Images
- Replace Unsplash URLs with your own images
- Update image paths in the HTML files
- Add your portrait to `images/portrait.jpg`
